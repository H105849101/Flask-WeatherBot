My Flask Weatherbot App

For this project I was asked to create a chatbot for the company GoTravel.

When running this app, run the main.py file. 

Chatbot Capabilities:
- This chatbot and answer various weather and forecast questions for any of the 10 places listed bellow:
  - Cumbria
  - Corfe Castle
  - The Cotswolds
  - Cambridge
  - Bristol
  - Oxford
  - Norwich
  - Stonehenge
  - Watergate Bay
  - Birmingham

- Examples of questions this chatbot can answer include:
  - What is the weather in Cumbria?
  - What is the forecast for Bristol in the next 5 days?
  - Do I need a jacket in Oxford tomorrow?
  - When is the sunset time at Stonehenge today?# Flask-WeatherBot
